# Exercise 004 — Conditionals and Loops

This exercise introduces control flow in Go.

You will learn how to make decisions with `if` statements and repeat work with `for` loops.

## Concepts Covered

- `if`, `else if`, and `else`
- basic comparison operators
- `for` loops
- looping with `range`
- using loops with slices

## Goals

In this exercise, the program should:

1. Check whether a person is an adult or a minor.
2. Print the numbers 1 through 5.
3. Loop through a list of names and print a greeting for each one.

## Expected Output

```text
Brad is an adult.
1
2
3
4
5
Hello, Brad
Hello, Alice
Hello, Sam
