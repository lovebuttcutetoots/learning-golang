# Exercise 002 — Variables and Basic Types

This exercise introduces Go variables and the most common built-in data types.

## Concepts Covered

- Variable declaration using `var`
- Short variable declaration using `:=`
- Basic Go types
- Printing variables
- String formatting with `fmt.Printf`

## Go Types Introduced

| Type | Description |
|-----|-----|
| string | Text values |
| int | Whole numbers |
| float64 | Decimal numbers |
| bool | True / false values |
