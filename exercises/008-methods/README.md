# Exercise 008 — Methods

This exercise introduces methods in Go.

Methods are functions that are attached to a specific type.

## Concepts Covered

- Defining methods on structs
- Method receivers
- Calling methods on struct instances

## Goals

In this exercise the program should:

1. Define a `Person` struct
2. Create a method that prints a person's information
3. Call the method on multiple `Person` instances

## Example Output

    Brad is 42 years old, lives in Las Vegas, and works as a loser.
    Alice is 30 years old, lives in Seattle, and works as an winner.
