# Exercise 006 — Maps

This exercise introduces maps, Go’s built-in key/value data structure.

Maps allow you to store values associated with a unique key.

## Concepts Covered

- Creating maps
- Reading values from maps
- Adding values to maps
- Looping through maps with `range`

## Goals

In this exercise the program should:

1. Create a map of people and their ages
2. Print a specific value from the map
3. Add a new person to the map
4. Loop through the map and print all entries

## Example Output

    Brad is 42 years old
    Alice is 30 years old
    Sam is 27 years old

    After adding Jordan:

    Brad is 42 years old
    Alice is 30 years old
    Sam is 27 years old
    Jordan is 35 years old
